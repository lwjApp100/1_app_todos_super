# 介绍

这是一个本地数据app,也就是说无需联网即可使用,整个app都与网络无关,如果不放心也可以直接在权限设置将此app的网络权限关闭
数据安全: 因为app内容都是地址信息和电话信息,所以数据都是直接存在本地手机上的(个人手机内存上),不会上传到网络
数据同步问题: 提供了数据导入导出功能
    导出: 可直接将当前所有数据导出成为一个很小的txt文件,只需自己保存好这个文件即可(保存在微信收藏、qq、其它地方都可)
    导入: 换设备后,只需选择之前保存的txt文件,然后选择导入,即可恢复之前的数据,也就是说,新导入的数据将会重新保存至当前设备的内存中


# 功能

    点完成历史查看历史完成过的事务
    在历史完成事务当中可重新恢复事务和彻底删除事务
